One-time setup 🔧

To share moments with everyone, this app stores data on JSONBin.io — a free shared database. You only need to do this once.

Step 1 — Get a free API key
Go to jsonbin.io → Sign Up → API Keys → copy your Secret Key.
Step 2 — Paste your Secret Key
Saved only in your browser's localStorage.
— then —
Step 3 — Bin ID (leave blank to create a new shared bin)
Share this ID with friends so you all see the same moments.
